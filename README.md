# MP3 Downloader
MP3 Downloader is a fast and lightweight Windows application designed to download multiple songs at once. It allows users to save large collections of MP3 files quickly, securely, and with high-quality audio output. The app is easy to use, safe, and optimized for both speed and performance, making it the perfect tool for anyone who frequently downloads music in bulk.
# Features
  1. Modern Hero Section with download button

  2. Fast & smooth UI designed with Bootstrap

  3. Fully responsive on all devices

  4. Features section showcasing app functionalities

  4. Guide section explaining how to install and use the app

  5. FAQ accordion for user support

  6. Footer with license information

  7. Blue gradient color theme for a clean modern look

# Technologies Used
  - HTML
  - CSS
  - BOOTSTRAP
