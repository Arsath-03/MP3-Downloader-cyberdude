# MP3 Downloader
A modern, fast, and user-friendly MP3 Downloader landing page built using HTML, CSS, and Bootstrap.
This project provides a clean UI for users to download the desktop version of the MP3 Downloader app, along with sections like features, guide, FAQ, and more.
# Features
  1. Modern Hero Section with download button

  2. Fast & smooth UI designed with Bootstrap

  3. Fully responsive on all devices

  4. Features section showcasing app functionalities

  4. Guide section explaining how to install and use the app

  5. FAQ accordion for user support

  6. Footer with license information

  7. Blue gradient color theme for a clean modern look

# Technologies Used
  - HTML
  - CSS
  - BOOTSTRAP
